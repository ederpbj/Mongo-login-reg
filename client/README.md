## Comandos no terminal

Descrição | Comando
------ | ------
__Comando__ | ------
Criar projeto ReactJS | `create-react-app client`
Ir a pasta client | `cd client`
Rodar projeto | `yarn start`
__Bibliotecas__ | ------
Axios e jwt | `npm i axios jwt-decode --save`
Router dom | `npm i react-router-dom --save` *
